## General info
This project is simple Typescript and nodejs app setup with nodemon and ava for test.

## Technologies
Project is created with:
* Typescript: `^4.3.2`
* node: `14`
* nodemon: `^2.0.7`
* ava: `^3.15.0`

## Setup
Commands to run : 
### Dev server
```javascript
npm run dev
```
### Dev server with debug
```javascript
npm run debug
```
### Run ava tests
```javascript
npm test
```
### Build nodejs app to dist folder
```javascript
npm build
```
